Help and Support

Pr_Suite Introduction ››

Pr_Suite provides informative help and support in the following ways:


  • Most controls provide tooltips. To view tool tips, hover your mouse pointer over desired control.



  • To find out the version and developer information for gizmos, hover your mouse pointer over ? icon.



  • Help tab for gizmos and help panel for python scripts explains purpose of gizmo/python script and provides Getting Started (Step by Step, Caution) information.



  • Support for gizmos provides direct access to Documentation, Tutorial, Report Bug and Check for Updates.



  • To directly contact developer for Requests and other Pr_Suite related issues e-mail at support@parimalvfx.com


  • For latest Pr_Suite updates and news visit http://www.parimalvfx.com/rnd/pr_suite/
